# Sprint 1 — Task 07: Errors and Validation

## 📄 Description

This repository contains the exercises for **Sprint 1 — Task 07**.

In this task, code robustness was improved by handling errors and exceptions, creating custom exceptions, and validating input using filters and regular expressions.

Each exercise is structured in its own folder with its corresponding classes and entry point.

---

## 🎯 Objectives

* Improve code robustness through error and exception handling
* Create custom exceptions
* Validate input using filters and regular expressions

---

## 🛠 Technologies

* PHP

---

## 🚀 Installation

Clone the repository:

```
git clone https://github.com/your-username/task-s1-07.git
```

---

## ▶️ How to Run

You can run each exercise independently using PHP's built-in server.

### Example:

```
php -S localhost:8000 -t level-1/exercise-1
```

Then open in your browser:

```
http://localhost:8000
```

👉 Replace the path with the exercise you want to run.
